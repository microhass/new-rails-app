## ğŸ’» Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need to have `ruby`, `rails` and `postgres` installed.

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone github.com/microhass/new_rails_app.git
```

Then navigate to `new_rails_app` folder:

```sh
cd new_rails_app
```

Install necessary dependencies:

```sh
bundle install
yarn install
```

Then start the app server:

```sh
rails serve
```

Then in another terminal:

```sh
yarn build --watch
```

Visit [http://127.0.0.1:3000/](localhost) to view the app.
